### What are databases?

Databases are structured systems used to store, organize, and manage data so that applications can access and use it efficiently.
They allow businesses to keep their information safe, consistent, and always available.Cloud engineers use databases to build scalable, secure, and reliable applications in the cloud.

### What do database professionals do?

Database professionals are responsible for setting up databases, modeling them, writing queries, and securing them.
